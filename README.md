# Pratham Chikitse 🏥

A life-saving offline first-aid emergency guide Android app
built for rural Karnataka communities.

## Features
- 🚨 20 Medical Emergency Protocols
- 🔊 Kannada Text-To-Speech Audio Guidance
- 📴 100% Offline — No Internet Required
- 🇮🇳 Bilingual — Kannada & English
- 🏥 Offline Hospital Finder
- 📞 One-tap 108 Ambulance Call

## Tech Stack
- Kotlin
- Android Studio
- ViewPager2
- Material Design 3
- Android TTS API
- RecyclerView
- Google AI Studio (GenAI Content)

## Project Purpose
Built as part of MindMatrix.io Android App Development
using GenAI Internship Program.

Pratham-Chikitse bridges the critical gap between the onset
of an emergency and arrival of professional medical help,
with medically accurate guidance in Kannada and English
for rural Karnataka communities.

## Min SDK
Android 8.0 (API 26) and above
